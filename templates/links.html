Chemistry and LaTeX

  • Chemfig - an extremely usefull LaTeX package for drwaing chemical structures. The basic knowledge of this package is a must.
  • Mol2chemfig - a python program that renders chemical structures from a molfile or SMILES format to chemfig syntax.
  • Mol2chemfig Web is a web application built upon Mol2chemfig package. Provides an user-friendly interface with several usefull features. A more detailed descpription can be found here.
  • TeX StackExchange is a website that can help if you get stucked and do not know how to solve problems related to LaTeX, do not hesitate to ask a question on TeX StackExchange./span>

Software used to develop Mol2chemfig Web

  • Flask is a webframework for Python. Minimalistic but very powefull. As name sugests, if you are chemist - use it.
  • ChemDoodle Web Components - a chemical graphics and cheminformatics library for the web. Easy to integrate and use. Has a rich set of functionalities but some services may require registration.
  • PubChemPy is a very handy python program for interaction with PubChem. Chemical searches by name, retrieval of chemical properties, chemical standardization are only few options out of many that PubChemPy provides.

Chemical Databases

If you need to get information about chemical and physical properties of a compound, check out some of the chemical databases: PubChem, ChemSpider, NIST Chemistry.
